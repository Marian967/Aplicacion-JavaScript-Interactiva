# Aplicacion-JavaScript-Interactiva
Trabajo Final Modulo 2

Descripción

Este proyecto fue desarrollado como trabajo final del módulo Introducción a JavaScript.
La aplicación web incluye tres funcionalidades principales:

•	Calculadora básica
•	Lista de tareas interactiva
•	Galería de imágenes

El objetivo del proyecto es aplicar conceptos fundamentales de JavaScript como:

•	manipulación del DOM
•	funciones
•	arreglos
•	eventos
________________________________________

Tecnologías Utilizadas

•	HTML5
•	CSS3
•	JavaScript
________________________________________

Funcionalidades:

Calculadora:
Permite realizar operaciones básicas como suma, resta, multiplicación y división.

Lista de Tareas:
Permite agregar, eliminar y marcar tareas como completadas.

Galería de Imágenes:
Permite seleccionar imágenes en miniatura para visualizarlas en tamaño completo.
